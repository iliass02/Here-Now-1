#Here & Now

- Installation à faire après avoir clone ce dépôt

    ```shell
    npm install -g cordova ionic
    cd here-and-now-app
    npm install
    bower install
    ```

- Installation émulateur iOS

    ```shell
    cd her-and-now-app
    ionic platform add ios
    ```

- Emulation App iOS

    ```shell
    cd here-and-now-app
    ionic build ios
    ionic emulate ios
    ```
